# <a name="compiler-error-cs0001"></a>Ошибка компилятора CS0001

Внутренняя ошибка компилятора.

 Возможно, ошибка возникает из-за сбоя компилятора при попытке проанализировать неправильный синтаксис. Если эта ошибка возникает снова, обратитесь в службу поддержки Майкрософт.
