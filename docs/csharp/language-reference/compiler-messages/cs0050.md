## <a name="example"></a><span data-ttu-id="f4ec2-107">Пример</span><span class="sxs-lookup"><span data-stu-id="f4ec2-107">Example</span></span>

 <span data-ttu-id="f4ec2-108">В следующем примере возникает ошибка CS0050. Это связано с тем, что для `MyClass` не указан модификатор доступности и по умолчанию используется уровень доступности `private`:</span><span class="sxs-lookup"><span data-stu-id="f4ec2-108">The following sample generates CS0050 because no accessibility modifier is supplied for `MyClass`, and its accessibility therefore defaults to `private`:</span></span>

```csharp

class MyClass // Accessibility defaults to private.
// Try the following line instead.
// public class MyClass
{
}

public class MyClass2
{
    public static MyClass MyMethod()   // CS0050
    {
        return new MyClass();
    }

    public static void Main() { }
}
```
