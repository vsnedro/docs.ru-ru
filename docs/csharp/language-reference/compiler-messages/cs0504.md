# <a name="compiler-error-cs0504"></a><span data-ttu-id="88c41-103">Ошибка компилятора CS0504</span><span class="sxs-lookup"><span data-stu-id="88c41-103">Compiler Error CS0504</span></span>
<span data-ttu-id="88c41-104">Константа "переменная" не может быть помечена модификатором static</span><span class="sxs-lookup"><span data-stu-id="88c41-104">The constant 'variable' cannot be marked static</span></span>  
  
 <span data-ttu-id="88c41-105">Если переменная имеет тип [const](../keywords/const.md), она также будет иметь модификатор [static](../keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="88c41-105">If a variable is [const](../keywords/const.md), it is also [static](../keywords/static.md).</span></span> <span data-ttu-id="88c41-106">Если вам требуется переменная типа **const** с модификатором **static**, достаточно просто объявить переменную как **const**. Если вам нужна просто переменная с модификатором **static**, пометьте ее как **static**.</span><span class="sxs-lookup"><span data-stu-id="88c41-106">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="88c41-107">В следующем примере возникает ошибка CS0504:</span><span class="sxs-lookup"><span data-stu-id="88c41-107">The following sample generates CS0504:</span></span>  
  
```csharp  
// CS0504.cs  
namespace x  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
