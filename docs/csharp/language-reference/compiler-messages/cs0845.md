# <a name="compiler-error-cs0845"></a><span data-ttu-id="fa852-103">Ошибка компилятора CS0845</span><span class="sxs-lookup"><span data-stu-id="fa852-103">Compiler Error CS0845</span></span>

<span data-ttu-id="fa852-104">Дерево лямбда-выражений не должно содержать оператор объединения с литералом null по левую сторону.</span><span class="sxs-lookup"><span data-stu-id="fa852-104">An expression tree lambda may not contain a coalescing operator with a null literal left-hand side.</span></span>  
  
 <span data-ttu-id="fa852-105">Поскольку значение NULL само по себе не имеет типа, к нему не может применяться оператор объединения со значением NULL.</span><span class="sxs-lookup"><span data-stu-id="fa852-105">Because null by itself does not have a type, the null coalescing operator cannot operate on it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="fa852-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="fa852-106">To correct this error</span></span>  
  
1. <span data-ttu-id="fa852-107">Приведите литерал NULL к объекту.</span><span class="sxs-lookup"><span data-stu-id="fa852-107">Cast the null literal to an object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fa852-108">Пример</span><span class="sxs-lookup"><span data-stu-id="fa852-108">Example</span></span>  

 <span data-ttu-id="fa852-109">В следующем коде возникает ошибка CS0845:</span><span class="sxs-lookup"><span data-stu-id="fa852-109">The following code generates CS0845:</span></span>  
  
```csharp  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```
