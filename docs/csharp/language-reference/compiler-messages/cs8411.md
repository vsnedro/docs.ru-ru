# <a name="compiler-error-cs8411"></a><span data-ttu-id="68439-102">Ошибка компилятора CS8411</span><span class="sxs-lookup"><span data-stu-id="68439-102">Compiler Error CS8411</span></span>

<span data-ttu-id="68439-103">Асинхронный оператор foreach не работает с переменными типа "type", так как "type" не содержит подходящее открытое определение экземпляра для "GetAsyncEnumerator"</span><span class="sxs-lookup"><span data-stu-id="68439-103">Asynchronous foreach statement cannot operate on variables of type 'type' because 'type' does not contain a suitable public instance definition for 'GetAsyncEnumerator'</span></span>

<span data-ttu-id="68439-104">Оператор `await foreach` работает только с типами, имеющими определение `GetAsyncEnumerator`, например `IAsyncEnumerable<T>`.</span><span class="sxs-lookup"><span data-stu-id="68439-104">`await foreach` statement operates only on types having a definition of `GetAsyncEnumerator`, such as `IAsyncEnumerable<T>`.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="68439-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="68439-105">To correct this error</span></span>

<span data-ttu-id="68439-106">Замените `await foreach` на `foreach`.</span><span class="sxs-lookup"><span data-stu-id="68439-106">Replace `await foreach` with `foreach`.</span></span>

## <a name="example"></a><span data-ttu-id="68439-107">Пример</span><span class="sxs-lookup"><span data-stu-id="68439-107">Example</span></span>

```csharp
using System.Collections.Generic;
using System.Threading.Tasks;

class Program
{
    async Task Example(IAsyncEnumerator<int> enumerator)
    {
        // error CS8411: Asynchronous foreach statement cannot operate on variables
        // of type 'IAsyncEnumerator<T>' because 'IAsyncEnumerator<T>' does not
        // contain a suitable public instance definition for 'GetAsyncEnumerator'
        await foreach (int i in enumerator)
        {
        }
    }
}
```
