# <a name="compiler-error-cs0156"></a><span data-ttu-id="d54e1-103">Ошибка компилятора CS0156</span><span class="sxs-lookup"><span data-stu-id="d54e1-103">Compiler Error CS0156</span></span>

<span data-ttu-id="d54e1-104">Недопустимо использовать оператор throw без аргументов в предложении finally, которая находится в ближайшем вложенном предложении catch.</span><span class="sxs-lookup"><span data-stu-id="d54e1-104">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="d54e1-105">Оператор [throw](../language-reference/keywords/throw.md) без параметров может использоваться только в предложении **catch** , которое не принимает параметров.</span><span class="sxs-lookup"><span data-stu-id="d54e1-105">A [throw](../language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="d54e1-106">Дополнительные сведения см. в разделе [исключения и обработка исключений](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="d54e1-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="d54e1-107">Следующий пример приводит к возникновению ошибки CS0156.</span><span class="sxs-lookup"><span data-stu-id="d54e1-107">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
