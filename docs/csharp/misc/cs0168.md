# <a name="compiler-warning-level-3-cs0168"></a>Предупреждение компилятора (уровень 3) CS0168

Переменная "var" объявлена, но не используется

Компилятор выдает предупреждение об уровне 3 при объявлении переменной, но не использует ее.
  
В следующем примере создается предупреждение CS0168:  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
