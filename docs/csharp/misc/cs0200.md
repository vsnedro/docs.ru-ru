## <a name="examples"></a><span data-ttu-id="0bab5-107">Примеры</span><span class="sxs-lookup"><span data-stu-id="0bab5-107">Examples</span></span>
<span data-ttu-id="0bab5-108">Следующий пример приводит к возникновению ошибки CS0200:</span><span class="sxs-lookup"><span data-stu-id="0bab5-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="0bab5-109">В следующем примере используются [свойства автоматического реализации,](../programming-guide/classes-and-structs/auto-implemented-properties.md) [инициализаторы объектов,](../programming-guide/classes-and-structs/object-and-collection-initializers.md)и по-прежнему генерирует CS0200:</span><span class="sxs-lookup"><span data-stu-id="0bab5-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md), [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md), and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="0bab5-110">Назначение на свойство или индексатор "имущество", которое читается только, может быть достигнуто путем добавления набора аксессуара или путем присвоения свойства в конструкторобъекта объекта.</span><span class="sxs-lookup"><span data-stu-id="0bab5-110">Assignment to a property or indexer 'property' that is read only, can be achieved through adding a set accessor or by assigning to the property in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
