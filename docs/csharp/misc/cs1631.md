# <a name="compiler-error-cs1631"></a><span data-ttu-id="e754c-103">Ошибка компилятора CS1631</span><span class="sxs-lookup"><span data-stu-id="e754c-103">Compiler Error CS1631</span></span>

<span data-ttu-id="e754c-104">Нельзя использовать оператор yield в теле предложения catch.</span><span class="sxs-lookup"><span data-stu-id="e754c-104">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="e754c-105">Использование оператора yield в теле предложения catch недопустимо.</span><span class="sxs-lookup"><span data-stu-id="e754c-105">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="e754c-106">Чтобы избежать этой ошибки, переместите оператор yield за пределы предложения catch.</span><span class="sxs-lookup"><span data-stu-id="e754c-106">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="e754c-107">Следующий пример приводит к возникновению ошибки CS1631:</span><span class="sxs-lookup"><span data-stu-id="e754c-107">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
