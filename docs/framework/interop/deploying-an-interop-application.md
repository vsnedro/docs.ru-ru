<span data-ttu-id="9916e-104">Приложения взаимодействия обычно включают клиентскую сборку .NET, одну или несколько сборок взаимодействия (представляющих различные библиотеки типов COM) и один или несколько зарегистрированных COM-компонентов.</span><span class="sxs-lookup"><span data-stu-id="9916e-104">An interop application typically includes a .NET client assembly, one or more interop assemblies representing distinct COM type libraries, and one or more registered COM components.</span></span> <span data-ttu-id="9916e-105">В Visual Studio и Windows SDK доступны средства для импорта и преобразования библиотеки типов в сборку взаимодействия, которые описываются в разделе [Импорт библиотеки типов в виде сборки](importing-a-type-library-as-an-assembly.md).</span><span class="sxs-lookup"><span data-stu-id="9916e-105">Visual Studio and the Windows SDK provide tools to import and convert a type library to an interop assembly, as discussed in [Importing a Type Library as an Assembly](importing-a-type-library-as-an-assembly.md).</span></span> <span data-ttu-id="9916e-106">Приложение взаимодействия можно развернуть двумя способами:</span><span class="sxs-lookup"><span data-stu-id="9916e-106">There are two ways to deploy an interop application:</span></span>

 <span data-ttu-id="9916e-115">Если вы используете встроенные типы взаимодействия, развертывание осуществляется очень просто.</span><span class="sxs-lookup"><span data-stu-id="9916e-115">If you use embedded interop types, deployment is simple and straightforward.</span></span> <span data-ttu-id="9916e-116">В этом случае никаких особых действий с вашей стороны не требуется.</span><span class="sxs-lookup"><span data-stu-id="9916e-116">There is nothing special you need to do.</span></span> <span data-ttu-id="9916e-117">Далее в этой статье описываются сценарии развертывания сборок взаимодействия с приложением.</span><span class="sxs-lookup"><span data-stu-id="9916e-117">The rest of this article describes the scenarios for deploying interop assemblies with your application.</span></span>
