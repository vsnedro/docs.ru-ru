# <a name="security-rules"></a>Правила безопасности

Правила безопасности поддерживают более безопасные библиотеки и приложения. Эти правила помогают предотвратить нарушение безопасности в программе. При отключении любого из этих правил следует четко отметить причину в коде, а также сообщить о назначенном начальнике системы безопасности для проекта разработки.
