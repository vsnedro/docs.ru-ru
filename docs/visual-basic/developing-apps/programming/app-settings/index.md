# <a name="accessing-application-settings-visual-basic"></a>Доступ к параметрам приложения (Visual Basic)

В этом разделе содержатся ссылки на разделы с описанием объекта `My.Settings` и задач, которые он позволяет выполнить.  
  
## <a name="mysettings"></a>My.Settings  

 Свойства объекта `My.Settings` обеспечивают доступ к параметрам приложения. Для добавления или удаления параметров используйте область **Параметры** **конструктора проектов**.  
  
 Методы объекта `My.Settings` позволяют сохранить текущие пользовательские параметры или восстановить последние сохраненные значения пользовательских параметров.
