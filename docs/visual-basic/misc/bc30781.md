# <a name="continue-must-be-followed-by-do-for-or-while"></a><span data-ttu-id="cca0f-102">За оператором "Continue" должны следовать операторы "Do", "For" или "While"</span><span class="sxs-lookup"><span data-stu-id="cca0f-102">'Continue' must be followed by 'Do', 'For' or 'While'</span></span>
<span data-ttu-id="cca0f-103">За оператором `Continue` должен следовать оператор `Do`, `For`или `While`в зависимости от того, находится ли оператор `Continue` внутри цикла `Do...Loop` , `For...Next` или `While...End While` .</span><span class="sxs-lookup"><span data-stu-id="cca0f-103">A `Continue` statement must be followed by `Do`, `For`, or `While`, depending on if the `Continue` statement appears within a `Do...Loop` loop, `For...Next` loop, or `While...End While` loop.</span></span>  
  
 <span data-ttu-id="cca0f-104">**Идентификатор ошибки:** BC30781</span><span class="sxs-lookup"><span data-stu-id="cca0f-104">**Error ID:** BC30781</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="cca0f-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="cca0f-105">To correct this error</span></span>  
  
1. <span data-ttu-id="cca0f-106">Если оператор `Continue` находится в цикле `Do...Loop` , измените оператор на `Continue Do`.</span><span class="sxs-lookup"><span data-stu-id="cca0f-106">If the `Continue` statement is in a `Do...Loop` loop, change the statement to `Continue Do`.</span></span>

3. <span data-ttu-id="cca0f-108">Если оператор `Continue` находится в цикле `While...End While` , измените оператор на `Continue While`.</span><span class="sxs-lookup"><span data-stu-id="cca0f-108">If the `Continue` statement is in a `While...End While` loop, change the statement to `Continue While`.</span></span>
