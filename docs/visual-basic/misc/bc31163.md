# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>Ожидалась соответствующая закрывающая кавычка для значения атрибута XML

В значении атрибута, которое разделяется одинарными кавычками, отсутствует закрывающая кавычка.  
  
 **Идентификатор ошибки:** BC31163  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Добавьте закрывающую одинарную кавычку ('), чтобы отделить значение атрибута.
